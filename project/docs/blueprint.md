# **App Name**: SparkFlow 24

## Core Features:

- Emergency Contact System: Prominent, floating 'Llamar ahora' and 'WhatsApp' buttons displayed across all pages for immediate user contact on any device, ensuring high conversion rates during emergencies.
- Dynamic Service Showcase: Clear and concise presentation of key emergency services (e.g., power outages, circuit breakers, wiring) using descriptive text and relevant icons for quick user comprehension.
- Trust & Credibility Display: Dedicated section highlighting professional credentials ('Electricistas matriculados'), 24/7 availability, rapid response commitment, work guarantees, and accepted payment methods.
- Simulated Customer Testimonials: Features three compelling, simulated customer reviews emphasizing service speed, effectiveness, and professionalism to build immediate social proof and confidence.
- Streamlined Contact Form: A simple, user-friendly contact form for non-urgent inquiries, requiring only name, phone, and message for efficient lead generation.
- Conversion-Optimized Layout: A clean, modern, and fully responsive page structure designed to instinctively guide users toward the main contact points across all device types, facilitating quick action during emergencies.
- Basic SEO Implementation: Integration of essential meta tags, descriptive titles, and optimized content structure to ensure fundamental search engine discoverability and quick loading times.

## Style Guidelines:

- Overall dark theme for a professional and serious tone. Main action elements like 'Llamar ahora' buttons use Electric Yellow (#FFFF00) for high visibility and urgency.
- The primary background is a very dark, deep blue (#161C26), embodying professionalism, trust, and aligning with the requested 'azul oscuro'.
- An analogous accent color, a soft muted orange (#E67252), is used sparingly for secondary highlights or illustrative elements to complement the vibrant yellow.
- Headlines use 'Space Grotesk' (sans-serif) for a modern, tech-informed and impactful look.
- Body text uses 'Inter' (sans-serif) for exceptional readability and a clean, neutral feel across various screen sizes.
- Linear, professional vector-based icons are used for service representations, colored white for clear contrast against dark backgrounds.
- A clean, multi-section layout with a fixed header, highly responsive design for optimal viewing and interaction on mobile devices, and strategically placed, always-visible CTAs.
- Subtle, smooth fade-in and slide-up animations for content sections on scroll, providing an elegant user experience without distracting from critical information or calls to action.
